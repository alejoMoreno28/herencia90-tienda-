# Matching Rubric

Use this rubric for every candidate image.

## Text Match

Score each reference before visual review:

- Team/country exact: required
- Year/season exact: required unless the source has a known equivalent label
- Variant exact: required
- Color exact: required
- Edition exact: required for centenario, player, manga larga, goalkeeper, special edition

If any required text field conflicts with the target reference, mark `REJECT`.

## Visual Match

Check the first image and then every gallery image:

- crest/team badge matches
- dominant color matches
- collar/stripe pattern matches expected reference
- sleeve length matches
- player/fan cut is plausible when required
- no mixed product from another listing
- no table/guide/banner/promo image
- no visible store handle or watermark

Decision labels:

- `APPROVE`: all required fields and images match.
- `PARTIAL`: some images match, some must be rejected.
- `NEEDS USER REVIEW`: good source but ambiguous year/edition.
- `REJECT`: mismatch, watermark, size chart, official render, or wrong product.

## Colombia Common Watchouts

- 1990 local is yellow; 1990 visitante is red.
- 1998 local is yellow; 1998 visitante is blue.
- 2024 local is yellow.
- 2024 away/visitante can be black or blue depending on listing. Do not merge them under one slug.
- Player version should not reuse fan version photos unless clearly marked as the same cut by the provider.
- 100 Anos / Centenario / Centenary should be a white special edition, not the normal yellow home.
